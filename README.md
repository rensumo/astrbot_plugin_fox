# 🐱 strbot_plugin_fox

一个AstrBot插件，可以随机抽取一张狐狸图并发送，同时@发送者。

## 🚀 功能介绍

- **随机抽取狐狸图**：从插件目录下的`fox`文件夹中随机抽取一张图片。
- **@发送者**：发送图片时自动@发送者。
- **使用**：发送指令`/狐狸图`或者`/🦊图`可触发插件功能。
- 插件小版本更新（0.0.X）是添加图片，建议按需更新
